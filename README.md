\[
\text{Market Rent (current month)} = \text{Annual Rent/SF} \times \left(1 + \text{Growth \%} \times \frac{\text{Month No.}}{12}\right)

- Current Month No. = **8**

**Full Service Gross (FSG):** All expenses included in rent, no additional reimbursements
